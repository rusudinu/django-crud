Welcome to my demo CRUD webpage, written in HTML, CSS and PYTHON [ + DJANGO ]. You can visit different pages of the website using the menu placed on the top of the page.

If you would like to interact with the website, you can use those links: CREATE, READ, UPDATE and DELETE. The links will redirect you to a page where you can interact with the SQL database.

CRUD stands for CREATE, READ, UPDATE and DELETE hence the buttons in the menu [ C | R | U | D ].
