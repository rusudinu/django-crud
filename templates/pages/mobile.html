DRIVERS DATA (+PRO): VIRTUAL LOGBOOK FOR DRIVERS

[ INTERNATIONAL ] [ AVAILABLE ] [ ANDROID ]

Drivers Data is an app coded in Java 8 for the Android platform, the main goal of this app being to create a virtual roadmap for all the drivers.Each user can get a report based on the entered data, where he can see a lot of parameters about his work days.Showcased at ILikeIT on Oct. 23, 2019. (ILikeIT is a show on PRO TV, the most popular TV station in Romania) Launched on Apr 28, 2019. Constant updates are released, to fulfill the user feature requests.


MYGOAL: EXPENSES ASSISTANT AND MONEY RAISER

[ INTERNATIONAL ] [ AVAILABLE ] [ ANDROID ]

This app aims at helping the user to raise a desired amount of money in the shortest timeframe possible. The user can enter the amount he wants to save, the amount he already havs and the amount he earns every month, to get an insight on how fast he'll be able to make his dream reality. Launched on Nov 26, 2019.


WHIST: SCORE TABLE

[ ROMANIA ONLY ] [ AVAILABLE ] [ ANDROID ]

This app helps the user track the score for the Whist game. It is intuitive and easy to use. No setup or account needed. All Whist game modes are supported. Online and Offline modes. Launched on Dec 15, 2019.


RUBRIKA (ANDROID)

GHICESTE PIESA

[ ROMANIA ONLY ] [ AVAILABLE ] [ ANDROID ]

This app is made for the romanian audience that listens to music. As of today, three flavors of this app exist: Ghicește Piesa, Ghicește Piesa - Piese de la AC/DC and Ghicește Piesa - Piese de la Metallica. The user gets the lyrics of a song in romanian and he has to guess the song title. Launched on March 08, 2020.
